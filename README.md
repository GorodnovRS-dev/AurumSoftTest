Для использования требуется собрать и запустить проект AurumSoftTest, загрузка и выгрузка данных осуществляется кнопками в интерфейсе.
Реализованы дополнительные пункты:
  •	Асинхронная загрузка файла с использованием async/await
  •	Экспорт сводной информации в JSON
  •	Написаны unit-тесты с использованием xUnit, проверяющие загрузку и валидацию данных
