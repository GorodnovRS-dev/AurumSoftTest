Для использования требуется собрать и запустить проект AurumSoftTest, загрузка и выгрузка данных осущетвляется кнопками в интерфейсе.
Релизованы дополнительные пункты:
  •	Асинхронная загрузка файла с использованием async/await
  •	Экспорт сводной информации в JSON
  •	Написаны unit-тесты с использованием xUnit, провереющие загрузку и валидацию данных
